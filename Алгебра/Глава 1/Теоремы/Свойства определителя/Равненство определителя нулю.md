*Определитель с двумя одинаковыми строками равен нулю*
$$
\Delta = \begin{vmatrix}
   a_{11} && a_{12} && ... && a_{1n} \\
   ... && ... && ... && ... \\
a_{i1} && a_{i2} && ... && a_{in} \\
... && ... && ... && ... \\
a_{i1} && a_{i2} && ... && a_{in} \\
... && ... && ... && ... \\
a_{n1} && a_{n2} && ... &&  a_{nn}
   \end{vmatrix} =^{(2)} - \begin{vmatrix}
   a_{11} && a_{12} && ... && a_{1n} \\
   ... && ... && ... && ... \\
a_{i1} && a_{i2} && ... && a_{in} \\
... && ... && ... && ... \\
a_{i1} && a_{i2} && ... && a_{in} \\
... && ... && ... && ... \\
a_{n1} && a_{n2} && ... &&  a_{nn}
   \end{vmatrix} = - \Delta
$$
$$
\Delta = - \Delta \ \Rightarrow 2\Delta = 0 \ \Rightarrow \Delta = 0.
$$

Доказательство аналогично пункту 4, в котором $c=0$.

*Определитель с двумя пропорциональными строками равен нулю*
$$
\begin{vmatrix}
   a_{11} && a_{12} && ... && a_{1n} \\ 
   ... && ... && ... && ... \\
   a_{i1} && a_{i2} && ... && a_{in} \\
   ... && ... && ... && ... \\
   ka_{i1} && ka_{i2} && ... && ka_{in} \\
   ... && .. && ... && ... \\
   a_{n1} && a_{n2} && ... && a_{nn}
   \end{vmatrix} =^{(4)} = k
   \begin{vmatrix}
   a_{11} && a_{12} && ... && a_{1n} \\
   ... && ... && ... && ... \\
   a_{i1} && a_{i2} && ... && a_{in} \\
   ... && ... && ... && ... \\
   a_{i1} && a_{i2} && ... && a_{in} \\
   ... && ... && ... && ... \\
   a_{n1} && a_{n2} && ... && a_{nn}
   \end{vmatrix}
   =^{(3)} k*0 = 0
$$

*Если в определителе одна из строк является линейной комбинацией других его строк, то такой определитель равен нулю*
   
Пусть в определителе $\Delta \ i$-я строка есть линейная комбинация $(i)=k_1(i_1)+k_2(i_2)+...+k_s(i_s).$

   \begin{vmatrix}
... \\
(i_1) \\
... \\
(i_2) \\
... \\
k_1(i_1) + ... + k_s(i_s) \\
... \\
(i_s)
...
   \end{vmatrix}
   =^{(7)}
   \begin{vmatrix}
   ... \\
   (i_1) \\
   ... \\
   (i_2) \\
   ... \\
   k_1(i_1) \\
   ... \\
   (i_s) \\
   ...
   \end{vmatrix}
   +
   \begin{vmatrix}
   ... \\
   (i_1) \\
   ... \\
   (i_2) \\
   ... \\
   k_2(i_2) \\
   ... \\
   (i_s) \\
   ...
   \end{vmatrix} + ... +
   \begin{vmatrix}

   ... \\

   ... \\
   k_s(i_s) \\
   ... \\
   (i_s) \\
   ...
   \end{vmatrix} =^{(6)} = 0+0+...+0=0
   $$
